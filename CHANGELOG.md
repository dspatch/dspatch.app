# Changelog

## 0.1.0

Initial release of d:spatch — Agent Orchestration Platform.

### Features

- Workspace management with visual agent hierarchy editor
- Agent lifecycle orchestration via Docker containers
- Real-time agent monitoring (logs, messages, events, files, usage)
- Inquiry system with priority-based routing and interrupt support
- Embedded HTTP/WebSocket server for agent-to-app communication
- Wire protocol with 26 typed packages (output, event, signal, connection)
- Agent template management with environment variables and volume mounts
- Workspace run history and replay
- Encrypted local database (Drift/SQLite)
- Hub integration for browsing and sharing agent/workspace templates
- Desktop app for Windows and macOS
